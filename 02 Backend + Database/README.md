## BUILD THE IMAGES
- Go with terminal in this folder and type: <br>
docker-compose build

## RUN THE MONGO CONTAINER
- Go with terminal in this folder and type: <br>
docker-compose up -d mongo

## RUN THE APP CONTAINER
- Go with terminal in this folder and type: <br>
docker-compose up -d app 
- Then go in the browser at the url: <br>
localhost:4000 

## STOP ALL CONTAINERS
- Go with terminal in this folder and type: <br>
docker-compose stop
